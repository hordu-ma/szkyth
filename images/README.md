## 图片格式说明
1. 所有图片建议使用以下格式：
   - PNG格式：适用于需要透明背景的图片，如logo、图标等
   - JPG格式：适用于照片类图片
2. 图片质量要求：
   - JPG图片压缩质量建议80-85%
   - 所有图片大小建议控制在300KB以内
3. 图片命名规范：
   - 使用小写字母
   - 用数字区分同类图片
   - 使用有意义的描述性名称
